# REST API CRUD App

This is a REST API learning basic mini project built using Node.js, Express and EJS.

## Features
- View all posts
- View single post
- Create new post
- Update post
- Delete post
- UUID used for unique IDs

## Tech Stack
- Node.js
- Express.js
- EJS
- UUID -package
